**Sistem Manajemen Kos**

Deskripsi Proyek

Sistem Manajemen Kos adalah aplikasi yang dibuat untuk membantu pemilik kos dalam mengelola data kamar, penyewa, pembayaran, dan laporan secara lebih terstruktur dan efisien.
Aplikasi ini dikembangkan sebagai bagian dari tugas kelompok mata kuliah Pola Desain Perangkat Lunak (PDPL).

Dengan sistem ini, pemilik kos dapat:
Mengelola data kamar kos
Mencatat data penyewa
Mengelola pembayaran sewa
Melihat laporan transaksi
Anggota Kelompok
